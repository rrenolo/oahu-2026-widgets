# oahu-2026-widgets

Two self-contained HTML widgets embedded in a private Notion trip planner,
hosted on GitHub Pages so Notion can iframe them.

- `weather.html` — official NWS forecast via api.weather.gov (free, no key)
- `tides.html` — NOAA CO-OPS tide predictions (free, no key)

No secrets, no build step, no dependencies. Source of truth: `~/Documents/Oahu-2026/assets/`.
